# Chalo Bus Pass System

A console-based Bus Pass Management System developed in C++ using Object-Oriented Programming and File Handling.

## Features

- User Registration
- User Login
- Bus Pass Plan Selection
- Balance Recharge
- Trip Tracking
- Pass Generation
- Data Storage using Files

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling

## How to Run

Compile:

g++ chalo_bus_pass_system.cpp -o buspass

Run:

./buspass

## Project Type

Academic Mini Project
